# ![logo](https://github.com/Kostya0Sim/NTMModernized-DM/blob/main/pack.png) NTM: Modernized DM
This resource pack changes the appearance of the [NTM Modernized](https://www.curseforge.com/minecraft/mc-mods/hbms-nuclear-tech-modernized) GUI to the darker and less saturated tones like in a [Default Dark Mode](https://www.curseforge.com/minecraft/texture-packs/default-dark-mode) resource pack.

Features:
- 1.20.1+ Version Support
- Machine GUI retexture (Assembler,Centrifuge,Cyclotron, etc.)
- NEI Support
- Bomb GUI retexture (Hydrogen Bomb, Fatman, Little Boy, Gadget, etc.)
- Reactor GUI retexture
- Turret GUI retexture
- Books and bobmazon retexture
